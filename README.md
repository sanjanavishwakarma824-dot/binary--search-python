# 🔍 Binary Search Algorithm in Python

This project demonstrates the implementation of Binary Search using Python.

## 📌 What is Binary Search?
Binary Search is an efficient searching algorithm used to find an element in a sorted list.

## ⚡ Features
- Fast searching (O(log n))
- Works on sorted arrays
- Beginner-friendly implementation

## 🛠️ How to Run

1. Run the Python file
2. Enter sorted numbers (space separated)
3. Enter the number you want to search
4. Get the index of the element

## 🧪 Example

Input:
2 4 6 8 10  
Target: 6  

Output:
Element found at index: 2
